⚙ N8N
Workflow
Engine
WF-1 Complaint Parser
  • Webhook trigger node
  • Validate: complaint_text not empty
  • Build OpenAI prompt (extraction)
  • OpenAI node → GPT-4o call
  • JSON parse structured output
  • Return: {category, duration, onset, symptoms[]}
WF-2 Flag Detector
  • Receives same payload as WF-1
  • Build flag detection prompt (10 conditions)
  • OpenAI node → GPT-4o (temp 0)
  • IF node: confidence ≥ 70%?
  • IF ≥ 90% → set urgency: critical
  • Return: {flags[], highest_confidence}
WF-3 Intake Submitter
  • POST /submit-intake trigger
  • Validate nurse_id present
  • Strip any non-confirmed fields
  • Write audit log (DB node)
  • EMR API write (V2 — stubbed V1)
  • Return: {encounter_id, status, timestamp}
WF-4 Queue Reader
  • GET /queue-state trigger
  • EMR read: active encounters
  • DB read: AI flag summary per encounter
  • Compute ESI distribution
  • Sort by acuity + wait time
  • Return: {esi_counts, patients[]}
3

🤖 OpenAI
Agent
GPT-4o
Prompt A — Structured Extraction GPT-4o · temp 0.2 · JSON mode
System
You are a clinical NLP assistant. Extract structured fields from ER triage chief complaints. Return JSON only. Never infer beyond what is stated. If field is unclear, return null.
Input
complaint_text: "Bee sting 5 min ago, throat feels tight, tongue swelling"
Output (JSON)
{ "complaint_category": "Insect sting / allergic reaction",
"onset": "acute — 5 minutes",
"symptoms": ["throat tightness", "tongue swelling"],
"trigger": "bee sting" }
Prompt B — Flag Detection GPT-4o · temp 0 · JSON mode
System
You are a critical flag detector for ER triage. Assess the following 10 conditions: stroke, acute MI, anaphylaxis, sepsis, respiratory distress, suicidal ideation, major trauma, hypertensive crisis, altered mental status, anaphylactoid reaction. Return confidence 0–100 and source_text citation. Do not surface flags below 70.
Input
complaint_text + vitals (BP, HR, SpO₂, RR)
Output (JSON)
{ "flags": [{ "condition": "anaphylaxis",
"confidence": 96,
"urgency": "critical",
"source_text": "throat feels tight, tongue swelling" }] }
Confidence Gate
Flag surfaces only if confidence ≥ 70%. Urgency banner triggers at ≥ 90%. Below 70 → silent drop.
Rejection Rule
If complaint_text < 10 chars or ambiguous → return "Insufficient detail" — no low-confidence guess.
Nurse Override
Nurse-confirmed field always wins. AI never overwrites a manually accepted value.
EMR Safety
AI output never touches EMR directly. All writes go through nurse sign-off on /submit-intake only.

Alert Confidence Routing
🔴 CRITICAL · Confidence ≥ 90%
Red pulsing alert card + urgency banner + audio indicator. Submit blocked until nurse explicitly confirms. Dismissal requires mandatory escalation note.
⚠ MODERATE · Confidence 70–89%
Amber alert card in sidebar. Source text citation shown. Nurse may accept or dismiss. Dismissed alerts logged with optional free-text reason.
— NO FLAG · Confidence < 70%
Silent. Flag is not surfaced. "Insufficient detail" shown if complaint too short to parse. No low-confidence guesses exposed to nurse.
